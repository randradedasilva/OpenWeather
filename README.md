
## Requirements

* Xcode 11 +
* Swift 5
* Cocoapods

## Design Pattern 

* Modular architecture with 3 layers: Domain, Infrastructure and Presentation
* Model-View-ViewModel (MVVM) with Coordinator pattern

## External libs 

* Quick / Nimble - For Behavior testing
* Alamofire - Networking 
* CodableAlamofire - Helper to decode Objects
* SDWebImage - Async lib to cache images 
* Reusable - Helper to register views more safely

## Installation
    
    git clone https://github.com/randradedasilva/OpenWeather.git
    
 Cocoapods install
 
 if you don't have cocoapods installed
  
    $ sudo gem install cocoapods
    
  if you already have cocoapods installed
  
    $ pod install
